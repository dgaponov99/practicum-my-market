# Бэкенд приложения Витрина магазина (реактивный стек)

### Требования:
- jdk 21
- docker engine

### Настройка окружения
- Указать данные для подключения к реляционной базе данных в `portal-ext.properties`

### Запуск тестов (модульные):
- Вызов `mvnw clean test`

### Запуск тестов (модульные и интеграционные):
- Запуск docker-engine
- Вызов `mvnw clean verify`

### Сборка jar:
- Вызов `mvnw clean package`
- Собранный fat-jar находится в `/target`

### Локальный запуск:
- Вызов `mvnw spring-boot:run`
- После запуска приложение доступно по `http://localhost`

### Тестовый запуск с тестовыми данными
- Запуск docker-engine
- Вызов `mvnw spring-boot:test-run`
- После запуска приложение доступно по `http://localhost`
